A smooth and addictive Flappy Bird-style desktop game built using pure HTML, CSS, and JavaScript. This project recreates the classic gameplay with clean visuals, responsive controls, and optimized performance for PC users.

✨ Features:

🎮 Simple keyboard controls (Spacebar to fly)
🐦 Smooth bird animation & gravity physics
🧱 Endless obstacle generation
🏆 Real-time score tracking
🔄 Game over & restart system
⚡ Lightweight and fast (No frameworks)

💻 Tech Stack:
HTML5 • CSS3 • JavaScript (Vanilla JS)

🚀 Perfect for beginners and developers who want to learn JavaScript game development or build fun browser-based games.
